StrongPlan

1RM 기반 스트렝스 트레이닝 루틴을 간단한 입력만으로 생성하는 정적 웹 도구입니다.

루틴 생성기 바로가기 스쿼트 영상 분석

제공 기능

  • 주간 스트렝스 루틴 생성(세트·반복·중량 가이드 포함)
  • 4대 운동 1RM 기반 중량 산출
  • 프로그램 선택을 위한 간단 가이드

사용 방법

  1. 프로그램 카드를 선택합니다.
  2. 스쿼트·벤치프레스·데드리프트·오버헤드프레스 1RM을 입력합니다.
  3. 훈련 경험과 주간 훈련 일수를 지정합니다.
  4. 루틴을 생성하고 결과 안내에 따라 중량을 해석합니다.

자주 묻는 질문

모바일에서도 사용 가능한가요?

예. 반응형 레이아웃으로 구성되어 휴대폰과 태블릿에서도 입력과 결과 확인이 가능합니다.

생성된 중량은 반드시 따라야 하나요?

표시 중량은 가이드입니다. 컨디션, 기술 숙련도, 장비 환경에 따라 조정하시기 바랍니다.

개인 정보는 저장되나요?

현재 버전은 브라우저 내에서만 계산하며 별도 저장이나 전송을 수행하지 않습니다.
